# Тестовое задание для комфорт софт
## Инструкция по сборке и запуску
1. Убедитесь что установлен JDK 17+ и Maven.
2. Клонируйте проект и перейди в корень.
3. Соберите командой в терминале:
    ```mvn clean package```
4. Запустите:
   ```java -jar target/test-comfort-soft-0.0.1-SNAPSHOT.jar```
5. Swagger UI находится по ссылке:
   [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)

6. Пример запроса:
   GET http://localhost:8080/min?path=input.xlsx&n=5


Конкретно для этого запроса был создан файл input.xlsx в корне проекта. Для проверки можно ввести свои данные в этот файл, 
или заменить этот файл своим с таким же названием.
В данном задании использовал зависимости для работы с swagger и xlsx файлами.
В качестве алгоритма для поиска числа выбрал Quickselect алгоритм O(n).
